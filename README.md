# ML Analytics Service

Akilli Sehir Yonetim Platformu icin tahmin ve karar destek servisidir.

## Sorumluluk

- IoT ve Video servislerinden REST API ile veri alir.
- Scikit-learn ile sicaklik ve kalabalik tahmini uretir.
- Rule-based risk engine ile risk skoru ve oneriler uretir.
- DynamoDB Local / Amazon DynamoDB uzerinde tahmin event'lerini saklar.

## Lokal Port

Docker Compose ile `8003` host portundan erisilir.
